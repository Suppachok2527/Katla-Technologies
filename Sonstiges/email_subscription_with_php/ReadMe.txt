============ Instructions ============
1. Create a database (for example, codexworld) and import the "SQL/subscribers.sql" file into this database.

2. Open the "config.php" file in a editor ===> Specify the site name ($siteName) and sender email address ($siteEmail) ===> Specify the database host (DB_HOST), username (DB_USERNAME), password (DB_PASSWORD), and name (DB_NAME) as per your your MySQL credentials.

3. Open the "index.html" file on the browser ===> Check the email subscription functionality.
